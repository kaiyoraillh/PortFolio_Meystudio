# CLAUDE.md

# Propósito del Proyecto

Este proyecto está enfocado exclusivamente en la creación de SITIOS WEB.

El usuario es principiante en programación web y quiere aprender mientras construye sitios reales.

Claude debe:

- Generar código claro y profesional
- Explicar brevemente las partes importantes
- Mantener el código simple
- Priorizar buenas prácticas de desarrollo web

Este proyecto NO está enfocado en:

- aplicaciones móviles
- scripts de sistema
- software de escritorio

Solo desarrollo web.

---

# Tecnologías Permitidas

Stack principal:

HTML5  
CSS3  
JavaScript (Vanilla)

Tecnologías opcionales cuando sea necesario:

Node.js  
Express.js  
LocalStorage  
APIs públicas

Evitar frameworks complejos si no son necesarios.

---

# Reglas de Código

Claude debe:

1. Escribir código limpio y legible
2. Usar nombres descriptivos
3. Mantener funciones simples
4. Agregar comentarios en el código
5. Evitar lógica innecesariamente compleja

Ejemplo de buenos nombres:

userMenu  
productCard  
submitForm  

Malos nombres:

x  
test1  
dataTemp  

---

# Estructura Estándar de Proyecto

Los sitios web deben organizarse de esta manera:

project/

index.html  
style.css  
script.js  

assets/
images/
icons/

components/
navbar.html
footer.html

pages/
about.html
contact.html

styles/
variables.css

scripts/
utils.js

---

# Reglas de Diseño Web

Los sitios deben ser:

- modernos
- responsive
- rápidos

Usar:

Flexbox  
CSS Grid  

Priorizar diseño mobile-first.

Evitar frameworks pesados de CSS.

---

# Modo Aprendizaje

Cuando Claude genere código debe:

1. Explicar brevemente el objetivo del código
2. Mostrar la estructura de archivos
3. Escribir el código completo
4. Incluir comentarios útiles
5. Explicar cómo probar el sitio

Las explicaciones deben ser cortas y fáciles de entender.

---

# Sistema de Presets

Claude puede usar diferentes presets dependiendo del tipo de sitio web solicitado.

El usuario puede escribir:

use preset: landing

o

use preset: blog

o

use preset: webapp

---

# PRESET: LANDING PAGE

Usar este preset para:

- páginas de producto
- páginas de marketing
- páginas de startups

Secciones recomendadas:

Hero  
Features  
Testimonials  
Call to Action  
Footer  

Objetivo:

diseño atractivo  
conversiones  
animaciones suaves

---

# PRESET: BLOG

Usar para sitios de contenido.

Estructura:

index.html  
post.html  

Características:

lista de artículos  
página de artículo  
menú de navegación  
tipografía optimizada para lectura

Opcional:

sistema de categorías

---

# PRESET: PORTFOLIO

Usar para sitios personales.

Secciones:

hero  
sobre mí  
proyectos  
habilidades  
contacto

Objetivo:

mostrar trabajos y habilidades.

---

# PRESET: WEB TOOL

Para herramientas web simples.

Ejemplos:

calculadoras  
generadores  
herramientas online

Características:

interactividad con JavaScript  
interfaz simple  
resultado en tiempo real

---

# Rendimiento

Claude debe:

- minimizar JavaScript
- optimizar CSS
- evitar librerías innecesarias

Las páginas deben cargar rápido.

---

# Seguridad Básica

Cuando haya formularios:

validar inputs  
sanitizar datos  
evitar exponer claves

---

# Cuando el Usuario Pida un Proyecto

Claude debe:

1. Identificar el preset
2. Mostrar la estructura del proyecto
3. Crear los archivos necesarios
4. Explicar cómo ejecutar el sitio

---

# Objetivo Final

El objetivo de este proyecto es:

aprender desarrollo web moderno mientras se crean sitios funcionales.
